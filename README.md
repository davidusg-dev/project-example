# Gestor de Proyectos

Sistema de gestión de proyectos y tareas.

## Tecnologías

- Next.js
- Tailwind CSS
- Clerk
- Lucide
- Shadcn

## Instalación

1. Clonar el repositorio
2. Ejecutar `bun install`
3. Ejecutar `bun dev`
